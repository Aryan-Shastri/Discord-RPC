## How to Use

1.  **Get the App**: Download the executable for your OS from the Releases page.
2.  **Get an App ID**:
    - Go to [Discord Developer Portal](https://discord.com/developers/applications).
    - Create a New Application.
    - Copy the **Application ID**.
3.  **Run the Tool**:
    - Paste the **Application ID**.
    - Fill in details (State, Details).
    - (Optional) Use Image Keys (upload images in Developer Portal -> Art Assets).
4.  **Connect**: Click **Connect** and **Update Presence**.
